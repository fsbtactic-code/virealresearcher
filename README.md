# Banana Parser — Stealth Архитектура Перехвата GraphQL (v3.1)

**Репозиторий:** [https://github.com/fsbtactic-code/virealresearcher](https://github.com/fsbtactic-code/virealresearcher)

Banana Parser — это профессиональный инструмент для скрытого скрапинга и анализа Instagram профилей, Reels, Каруселей и хэштегов. Главная особенность: полный отказ от парсинга HTML DOM. Инструмент перехватывает GraphQL и JSON трафик на уровне сети через `playwright` stealth mode, что сводит вероятность блокировок профиля к минимуму и многократно ускоряет отдачу данных.

## Возможности

- **Пассивный сетевой перехват**: Прямой перехват запросов (XHR/Fetch) к API Instagram (без `<div class="...">` и BeautifulSoup).
- **Stealth-инъекции**: Поддержка `playwright-stealth` для имитации живого браузера. Совершенно прозрачная работа с UI Instagram.
- **Интеграция Claude MCP**: Нативная интеграция навыков Claude Code. Управление скриптом прямо из IDE как через MCP Server.
- **Аналитика Виральности**: Оценка Velocity Score (лайков/ч) и фильтрация нулевого вовлечения.
- **UI на базе Webview**: Интерфейс на Liquid Glass (CSS/JS) встроенный прямо в Python-процесс через `pywebview`. Отвязан от системных браузеров.
- **Автоматический парсинг Subtitles**: Резервная система Whisper для "немых" Reels (поддержка AI-классификации).

## 🚀 Автоматическая Установка через Claude Code

Вам не нужно вводить команды вручную. **Просто скопируйте нужный промпт ниже**, вставьте его в `Claude Code` и нажмите Enter. Claude сам скачает репозиторий, настроит окружение, установит зависимости, интегрирует MCP сервер и подключит все скиллы (banana-run, banana-auth и др.).

### Промпт для Windows:
```text
Установи проект Banana Parser. Действуй на 100% автономно:
1. Склонируй https://github.com/fsbtactic-code/virealresearcher.git (если папки нет) и перейди в нее.
2. Создай виртуальное окружение: `python -m venv .venv` и активируй его с помощью `.venv\Scripts\activate`.
3. Установи зависимости: `pip install -r requirements.txt`.
4. Запусти `python install_all.py` (он скачает Playwright и необходимые модели).
5. Просканируй папку .claude/skills и интегрируй все навыки в мой глобальный конфиг `~/.claude.json`. Пропиши абсолютные пути к mcp_server.py для MCP.
6. В конце скажи "Установка завершена! Введите /banana-run для старта".
Не задавай вопросов, выполняй команды в PowerShell последовательно.
```

### Промпт для macOS / Linux:
```text
Установи проект Banana Parser. Действуй на 100% автономно:
1. Склонируй https://github.com/fsbtactic-code/virealresearcher.git (если папки нет) и перейди в нее.
2. Создай виртуальное окружение: `python3 -m venv .venv` и активируй его: `source .venv/bin/activate`.
3. Установи зависимости: `pip install -r requirements.txt`.
4. Запусти `python3 install_all.py` (установит браузеры Playwright).
5. Просканируй директорию .claude/skills и интегрируй все скиллы в мой глобальный профиль. Убедись, что MCP сервер прописан в конфиг `~/.claude.json`.
6. Выведи: "Установка успешно завершена! Введите /banana-run для запуска."
Предоставь мне готовое решение, выполняй все bash-команды автоматически.
```

Если вы не используете Claude Code, подробные шаги для разработчиков описаны в файлах `INSTALL_PROMPT_WIN.md` и `INSTALL_PROMPT_MACOS.md`.

## Основные Файлы Архитектуры

* `interceptor.py` — Ядро перехвата GraphQL/XHR трафика.
* `run_scraper.py` — Главный воркер и API-мост для UI.
* `mcp_server.py` — stdio сервер для интеграции Banana Parser в Claude Desktop.
* `ui_generator.py` — Генератор интерактивных HTML-отчетов.

Подробная архитектура описана в файле `.ai-context.md`.
