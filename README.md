## 🚀 Установка через Claude AI (самый быстрый способ)

Скопируйте этот промпт целиком и вставьте в **Claude Code** (или любой AI-агент с терминалом):

```text
Установи и настрой FilPars:
1. git clone https://github.com/fsbtactic-code/virealresearcher.git (если папка уже есть — зайди в неё и сделай `git pull` для обновления).
2. Скопируй все скиллы из скачанной папки `virealresearcher/.claude/skills/` в папку `.claude/skills/` ТЕКУЩЕГО ПРОЕКТА (откуда ты вызван).
3. Обнови `settings.json` текущего проекта: явно включи скопированные скиллы в `skillOverrides` и дай разрешения на выполнение `Bash(*)`.
4. Найди глобальный файл конфигурации Claude (`~/.claude.json` или `%APPDATA%\Claude\claude_desktop_config.json`) и автоматически пропиши туда наш MCP-сервер `banana-parser` с АБСОЛЮТНЫМ ПУТЕМ к `virealresearcher/mcp_server.py`.
5. Перейди в скачанную папку `virealresearcher` и выполни `python install_all.py` (если будут ошибки — проанализируй и исправь).
6. Попроси меня запустить `/filparsauth` чтобы открыть окно для входа в Instagram.
7. Когда я скажу, что авторизовался (и окно само закроется), попроси меня запустить парсер через `/filparsrun`.

При любых непонятных ошибках проекта читай файл `virealresearcher/.ai-context.md`.
```

**Всё.** AI-агент сам установит зависимости, откроет окно для защищенного входа в Instagram и запустит интерфейс.

## 💡 Что это такое?

FilPars — это **десктопное приложение**, которое автоматически:

- 🔍 **Сканирует Instagram** — ленту, explore, поиск по ключевым словам
- 📊 **Анализирует вирусность** — рассчитывает velocity score, engagement rate, потенциал роста
- 🔥 **Находит «горячие» посты** — выделяет контент с аномально высокими метриками
- 📋 **Экспортирует в Google Sheets** — один клик и данные у вас в таблице

### Зачем это нужно?

Вместо ручного скроллинга Instagram вы получаете **автоматизированный разведчик контента**. Он собирает сотни постов конкурентов, фильтрует по вашим критериям и показывает только то, что реально «залетает».

## 🖥️ Как это работает

```
1. Запустите приложение
   └── Откроется нативное окно с настройками

2. Настройте параметры
   ├── Ключевые слова (через запятую)
   ├── Глубина поиска (6h / 12h / 24h / 48h)
   ├── Типы контента (Reels, Карусели, Фото)
   └── Минимальные лайки, фолловеры

3. Нажмите «НАЧАТЬ ПАРСИНГ»
   ├── Парсер открывает headless-браузер
   ├── Перехватывает данные из GraphQL API Instagram
   ├── Фильтрует и ранжирует посты в реальном времени
   └── Показывает прогресс: "Собрано: X / Подходит: Y"

4. Получите результаты
   ├── 🔥 Карточки «горячих» постов
   ├── 📊 Сортируемая таблица всех результатов
   ├── 📋 Экспорт в CSV / Google Sheets
   └── 🔗 Прямые ссылки на каждый пост
```

## 🧠 MCP-сервер (для Claude Code)

FilPars также может работать как **MCP-сервер** — Claude вызывает парсинг через tool calls:

```json
{
  "mcpServers": {
    "banana-parser": {
      "command": "python",
      "args": ["C:/АБСОЛЮТНЫЙ/ПУТЬ/К/ПАПКЕ/virealresearcher/mcp_server.py"],
      "cwd": "C:/АБСОЛЮТНЫЙ/ПУТЬ/К/ПАПКЕ/virealresearcher"
    }
  }
}
```
*Внимание: путь в `args` должен обязательно включать полный абсолютный путь к файлу mcp_server.py, иначе Claude может искать файл в системных папках!*
