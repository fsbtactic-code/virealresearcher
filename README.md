## 💡 Что это такое?

Banana Parser — это **десктопное приложение**, которое автоматически:

- 🔍 **Сканирует Instagram** — ленту, explore, поиск по ключевым словам
- 📊 **Анализирует вирусность** — рассчитывает velocity score, engagement rate, потенциал роста
- 🔥 **Находит «горячие» посты** — выделяет контент с аномально высокими метриками
- 📋 **Экспортирует в Google Sheets** — один клик и данные у вас в таблице

### Зачем это нужно?

Вместо ручного скроллинга Instagram вы получаете **автоматизированный разведчик контента**. Он собирает сотни постов конкурентов, фильтрует по вашим критериям и показывает только то, что реально «залетает».

## 🖥️ Как это работает

```
1. Запустите приложение
   └── Откроется нативное окно с настройками

2. Настройте параметры
   ├── Ключевые слова (через запятую)
   ├── Глубина поиска (6h / 12h / 24h / 48h)
   ├── Типы контента (Reels, Карусели, Фото)
   └── Минимальные лайки, фолловеры

3. Нажмите «НАЧАТЬ ПАРСИНГ»
   ├── Парсер открывает headless-браузер
   ├── Перехватывает данные из GraphQL API Instagram
   ├── Фильтрует и ранжирует посты в реальном времени
   └── Показывает прогресс: "Собрано: X / Подходит: Y"

4. Получите результаты
   ├── 🔥 Карточки «горячих» постов
   ├── 📊 Сортируемая таблица всех результатов
   ├── 📋 Экспорт в CSV / Google Sheets
   └── 🔗 Прямые ссылки на каждый пост
```

## 🔧 Ручная установка

Если хотите установить без AI-агента:

```bash
# 1. Клонируйте
git clone https://github.com/banana-marketing/banana-parser.git
cd banana-parser

# 2. Установите зависимости
pip install -r requirements.txt
python -m playwright install chromium

# 3. Авторизуйтесь (один раз)
python auth.py

# 4. Запустите
python run_scraper.py
```

## ⚠️ Важно

> **Используйте ТОЛЬКО технический аккаунт Instagram.**
>
> Instagram может временно ограничить аккаунт, используемый для автоматического сбора данных. Создайте отдельный аккаунт, прогрейте его 2-3 дня, подпишитесь на аккаунты вашей ниши.

## 📂 Структура проекта

```
banana-parser/
├── run_scraper.py        # Точка входа
├── skills.py             # Логика парсинга + формула виральности
├── interceptor.py        # Network interception + фильтрация
├── browser_core.py       # Stealth Playwright
├── auth.py               # Авторизация Instagram
├── web_launcher.py       # pywebview (нативное окно)
├── ui_generator.py       # Генератор HTML-отчётов
├── mcp_server.py         # MCP-сервер для Claude Code
├── install_all.py        # Универсальный установщик
├── requirements.txt      # Python-зависимости
└── ui_templates/
    └── launcher.html     # Интерфейс (настройки + дашборд)
```
