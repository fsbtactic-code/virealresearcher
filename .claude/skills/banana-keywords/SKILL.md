# /banana-keywords — Составление Точного Списка Ключевых Слов для Ниши

## Что делает этот скилл

Помогает пользователю составить **прецизионный список ключевых слов** для фильтрации постов по теме. При правильном списке парсер будет собирать только релевантные посты и не пропустит ни одного важного — без мусора и двусмысленностей.

## Когда использовать

Когда ты хочешь настроить ИИ распознавание тематики в Banana Parser и не знаешь, какие конкретные слова выбрать для своей ниши.

## Алгоритм работы

1. Спроси у пользователя описание его ниши (1-3 предложения: чем занимается, кто ЦА, что ищет)
2. Проанализируй нишу по трём уровням специфичности:
   - **Tier 1 — Бренды и инструменты**: конкретные продукты, компании, имена (Claude, Midjourney, Notion)
   - **Tier 2 — Профессиональные термины**: специализированная лексика ниши (LLM, промпт, файнтюнинг)
   - **Tier 3 — Контекстные слова**: слова, которые *в контексте ниши* однозначны (нейросеть, автоматизация, агент)
3. Составь список из 20-50 слов, исключив:
   - Слишком широкие (AI, digital, маркетинг — если это не конкретная ниша)
   - Двусмысленные (Python — это язык программирования ИЛИ питон?)
   - Слишком редкие (попадётся 1 пост в неделю)
4. Выдай список в формате, готовом для вставки в поле фильтра Banana Parser

## Формат вывода

```
КЛЮЧЕВЫЕ СЛОВА ДЛЯ НИШИ: [название]
Готово к вставке в Banana Parser → поле "ИИ распознавание тематики"

[список через запятую, одна строка]

---
ОБЪЯСНЕНИЕ ВЫБОРА:
Tier 1 (бренды): [список]
Tier 2 (термины): [список]
Tier 3 (контекст): [список]

ИСКЛЮЧЕНО И ПОЧЕМУ:
- [слово] — [причина]
```

## Пример запроса

> `/banana-keywords` Я занимаюсь обучением нейросетям для бизнеса. Продаю курсы и консультации по внедрению AI. Аудитория — предприниматели и маркетологи.

## Результат

Готовый список для вставки в Banana Parser без необходимости ручного подбора и тестирования. Список можно обновлять при изменении ниши.

## Связанные скиллы

- `/banana-run` — запуск парсера с настроенными ключами
- `/banana-debug` — диагностика если парсер не находит посты по теме
